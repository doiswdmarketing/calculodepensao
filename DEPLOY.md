# Homologação - Calcula Pensão

Esta pasta contém somente os arquivos necessários para publicar a landing page em ambiente de homologação.

## Arquivos principais

- `index.html`
- `styles.css`
- `script.js`
- `design-tokens.css`
- `assets/`

## Observações

- O botão "Iniciar simulação" aponta para:
  `https://calculapensao.com.br/checkout/?add-to-cart=428`
- Esta pasta não inclui os arquivos internos do projeto, como Design System, copy original e fotos fonte.
